# SkillPath – Skill Progress Tracking System

SkillPath is a Flask-based web application that helps learners track skill development, 
set milestones, generate to-do lists, track time, and view analytics dashboards.

### Features (Phase-1):
✔ User Registration/Login  
✔ Skill Management (add skill, milestones, deadlines)  
✔ Progress Tracking  
✔ Auto To-Do List Generation  
✔ Notifications (Phase-2)  
✔ Reports (PDF)
